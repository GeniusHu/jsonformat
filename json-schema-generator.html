Frequently Asked Questions

What is JSON Schema?

JSON Schema is a vocabulary that allows you to annotate and validate JSON documents. It helps you define the structure, types, and validation rules for your JSON data.

How accurate is the type inference?

Our type inference is highly accurate for common data types and patterns. It can detect strings, numbers, booleans, arrays, and objects, as well as special formats like dates and emails.

Can I customize the generated schema?

Yes, you can customize the generated schema by adding validation rules, modifying property requirements, and adjusting type definitions through our user interface.
